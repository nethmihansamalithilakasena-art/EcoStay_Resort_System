 EcoStay Core - Premium Infrastructure Management Deck

A high-performance, modular Resort Management System built with **Java (Swing/WPF styling philosophy)** and **MySQL** database frameworks.

## 🚀 Enterprise Architecture Features
* **Role-Based Security Portal:** Clean login gateway separating administrative managers from reception workflows.
* **ACID Financial Ledger:** Dynamic real-time calculation matrices managing advance deposits and balance tracking.
* **Live Audit Inspector:** Dual-pane split viewport displaying rich text guest profiles and historical transaction timelines.
* **Dynamic Table Matrix Toggle:** In-memory view sorter allowing operational logs to flip between descending chronological orders instantly.
* **Jasper Report Printing Engine:** Automated compilation scripts for generating pixel-perfect corporate financial registries.

## 🛠️ Tech Stack & Dependencies
* **Core Language:** Java JDK 17+
* **Database Driver:** MySQL Connector/J 8.0+
* **UI Look and Feel:** FlatLaf Light Theme Component Suite
* **Reporting Architecture:** JasperReports Core Engines 6.x+
* **Build Automation:** Maven Project Wrapper (`pom.xml`)

## 💾 Database Setup Initialization
To initialize the relational schemas, import the `database.sql` script located at the project root directory straight into your active local MySQL Server instance.
